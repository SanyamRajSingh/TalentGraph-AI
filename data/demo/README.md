# Demo Dataset

This folder contains a compact hackathon demo set for Module 7.

Recommended flow:

1. Paste `job-description.md` into Step 1.
2. Paste each resume into Step 2 and build candidate twins.
3. Evaluate each candidate against the generated role.
4. Rank candidates with the Startup Founder persona.
5. Generate explanations for ranked candidates.
6. Export rankings to XLSX.

- `job-description.md`
- `resumes/01-mira-shah.md`
